# MY-TO-DAY

## Overview

나의 하루 일과를 기록하는 앱입니다.


## Prerequisite

|    NAME     | VERSION |
| :---------: | :-----: |
| Flutter SDK |  3.7.7  |


## Project Stack

- Flutter
- Fvm
- Provider
- Hive
- Go_router
- google maps

## Getting started

해당 프로젝트는 Flutter SDK 버전 관리 툴인 fvm을 사용하고 있으므로
fvm을 적용하여 프로젝트에 적용 된 SDK 버전으로 세팅해야합니다.
